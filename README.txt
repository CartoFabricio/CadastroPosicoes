Teste Técnico Imagem: Sistema para Cadastro de Posições


Esse sistema realiza o cadastro ou agendamento de alugueis para salas, mesas ou espaços compartilháveis. Para executar o sistema, selecione o arquivo .exe, localizado na pasta CadastroPosicoes. Aguardo enquanto o sistema é inicializado, após, realize o login ou cadastro de novo usuário ou a busca de espaços por meio da tela inicial. Aplique os filtros (ex.: data, materiais) para melhor descrever seus requerimentos básicos.
